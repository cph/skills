# Product Developer


#### Technical Portfolio

This repo lists the languages, tools, frameworks, and techniques we rely on most heavily in Emerging Products. For each, it gives two things:

 * Roughly what the path to mastery looks like
 * Resources we've found helpful for leveling up along the way


#### Five Milestones of Skill Acquisition

<table>
  <tr><th valign="top" align="left">Novice</th><td><ul>
    <li>Unaware of context</li>
    <li>Must follow context-free rules</li>
  </ul></td></tr>
  
  <tr><th valign="top" align="left">Advanced Beginner</th><td><ul>
    <li>Recognizes aspects of context</li>
    <li>Can start applying context-sensitive rules</li>
    
    <li>Has <em>memorized</em>the grammar</li>
  </ul></td></tr>
  
  <tr><th valign="top" align="left">Competent</th><td><ul>
    <li>Appreciates breadth of context; analyzes it slowly</li>
    <li>Applies rules of thumb</li>
    
    <li>Has <em>understanding</em> of the internals</li>
  </ul></td></tr>
  
  <tr><th valign="top" align="left">Proficient</th><td><ul>
    <li>Has <a href="http://en.wikipedia.org/wiki/Memoization">memoized</a> analysis of very many contexts</li>
    <li>Makes decisions consciously</li>
    
    <li>Has <em>applied experience</em> of most contexts</li>
  </ul></td></tr>
  
  <tr><th valign="top" align="left">Expert</th><td><ul>
    <li>Acts intuitively</li>
    
    <li>Instinctively grasps trade-offs</li>
  </ul></td></tr>
</table>
